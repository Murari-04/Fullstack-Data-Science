# 🏠 House Price Prediction

A Machine Learning web application that predicts house prices using multiple Regression algorithms. The project compares different models and uses the best-performing model for prediction.

## 📌 Features

- Predicts house prices based on input features
- Compares multiple Regression algorithms
- User-friendly Flask web interface
- Saves trained models as `.pkl` files
- Displays model evaluation results

## 🛠️ Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- LightGBM
- HTML & CSS

## 📁 Project Structure

```
House Price Prediction/
│── templates/
│── app.py
│── model.py
│── *.pkl (Trained Models)
│── model_evaluation_results.csv
│── README.md
```

## 🚀 How to Run

1. Install the required libraries.
2. Run:

```bash
python app.py
```

3. Open the application in your browser and enter the house details to get the predicted price.

## 📊 Output

The application predicts the estimated house price using the best-performing regression model.

## 📊 Machine Learning Workflow

1. Load the dataset
2. Clean and preprocess the data
3. Encode categorical variables
4. Split the dataset into training and testing sets
5. Train the regression model
6. Evaluate model performance
7. Save the trained model
8. Predict house prices from user inputs
